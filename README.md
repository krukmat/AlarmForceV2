# Alarmforce v2.0
## Description: 
-------------------
The framework is based on IOTForce stack:
* Features:
    * Remote alarm monitoring
    * Notification when any suspicious activity is detected
    * Wifi less communication based on LORA protocol
* IOT Proxy:
    * Ioticos(MQTT)
*  Hardware:
    * ESP32 Camera
    * 2 Lora-TTGO
    * Noise sensor
* Software:         
    * Arduino IDE
    * Salesforce:
